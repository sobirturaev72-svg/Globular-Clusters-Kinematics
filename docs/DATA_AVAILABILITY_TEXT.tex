All kinematic source data are publicly available through the HACKS HLSP. The archived release contains the standardized 20-cluster radial profiles, the primary and full-range KAP, GKI, $D_s$, and $D_{s,\rm corr}$ tables, radial-coverage and NGC~6388 audit products, figure-generation scripts, archived figures, and numerical validation checks. Third-party source catalogues are not redistributed in full; provenance and source links are documented in the release.
