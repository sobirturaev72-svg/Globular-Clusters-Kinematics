# Zenodo upload checklist

1. Upload the single ZIP archive, not the extracted folder.
2. Use the title and creator order from `.zenodo.json`.
3. Select open access and CC BY 4.0 for the release metadata.
4. Do not add the manuscript PDF or TeX source to this record.
5. Confirm version `1.1.0` and publication date before publishing.
6. After Zenodo assigns the DOI, add it to the article Data Availability section.
7. If GitHub-Zenodo integration is used, create a matching `v1.1.0` Git tag/release.
8. Verify the deposited ZIP checksum against the value reported beside the download link.
