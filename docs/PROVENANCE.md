# Provenance

## HACKS kinematic source

- HLSP: Hubble Space Telescope Atlases of Cluster Kinematics (HACKS)
- Landing page: https://archive.stsci.edu/hlsp/hacks
- Source DOI: 10.17909/jpfd-2m08
- All-cluster velocity-dispersion product: recorded in `data/source_inventory/hacks_source_record.json`
- Retrieved for source audit: 2026-07-15

The archive does not redistribute the all-cluster source file. It includes the
standardized 20-cluster derived table used in the analysis and a source-product
inventory. Structural and orbital properties are documented in the data
dictionary and originate from the cited catalogues/databases used by the article.

## Derived layer

KAP, GKI, analytic uncertainties, profile-scatter diagnostics, radial-coverage
quantities, correlations, article tables, and figures were generated from the
standardized profile and property tables by the packaged scripts.

Associated article: **Projected Kinematic Anisotropy and Velocity-Dispersion Gradients in Galactic Globular Clusters: An Interval-Defined Framework from HST Proper Motions**.
