# Synthetic validation status

## Reported article reference summaries

`results/validation/synthetic_validation/tests_A_to_E_reported_reference.csv`
and `test_F_reported_reference.csv` preserve the rounded values reported in the
article. The original random-state information and intermediate realisation
arrays were not available in the source bundle used to assemble this release.
These files are therefore explicitly labelled `reported_reference_only` and are
not claimed to be bitwise-reproducible stochastic outputs.

## Reproducible release components

The observational point estimates, analytic uncertainties, interval
classifications, radial-coverage audit, KAP-normalized-anisotropy identity,
leave-one-bin-out calculations, vector figures, and the fixed-seed 10,000-draw
parametric rerun are reproducible from the packaged data and scripts.

## Interpretation

The synthetic references are supporting checks, not an independent confirmatory
layer. The article's principal claims rely on the audited observational tables
and the interval-normalized analysis. No synthetic summary is used to alter a
cluster classification.
