# MATLAB figure renderer

Run MATLAB from this directory:

```matlab
run('START_HERE.m')
```

The scripts read the archived input tables in `data/input/` and regenerate the
seven manuscript figures as vector PDF files in `results/figures/pdf/`. Figure
windows remain open after export. MATLAB R2021a or later is recommended; no
third-party toolbox is required.

The DOI archive intentionally stores only the vector PDF figure products. It
does not duplicate them as PNG, EPS, or MATLAB FIG files.
